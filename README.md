# 寝室の置き時計

寝室などで活用できるスクリーンセイバー
古いスマホを利用できるように、Android 6.0以上としました。


## 主な機能

- スクリーンセイバ
- 昼夜（6:00,18:00）の自動切り替え
- 昼間は青空のグラデーション
- 夜は星がきらめく空
- 時計は極力大きくし、夜でも見やすくしました
- アラーム設定がある場合は、時計の下に表示

## 技術スタック

| 項目 | 使用技術 |
|---|---|
| 言語 | Kotlin |
| UI | RecyclerView, SearchView, SeekBar |
| 最小SDK | API 23 (Android 6.0) |
